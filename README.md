# Seating Chart – Event Check-In System

## Overview

The Seating Chart – Event Check-In System is a web-based application designed to streamline guest verification and table assignment during events. The system enables organizers to efficiently check in attendees, display assigned tables, and monitor real-time attendance updates.

This project was built to improve event-day coordination by minimizing manual tracking and reducing delays during guest arrival.

---

## Features

- Guest search and verification  
- Automated table assignment display  
- Real-time check-in status updates  
- Google Sheets integration for live data storage  
- User-friendly interface designed for event staff  

---

## System Architecture

The system follows a lightweight web-based architecture:

- **Frontend:** Handles guest lookup, table display, and check-in interface  
- **Backend (Google Apps Script):** Processes requests and manages application logic  
- **Database (Google Sheets):** Stores guest list, table assignments, and check-in status  

This structure allows for easy deployment, scalability, and minimal maintenance.

---

## Technologies Used

- HTML

---

## How It Works

1. Guests open the web interface.
2. Guests search for their name.
3. The system retrieves the assigned table from Google Sheets.
4. Upon confirmation, the check-in status is updated in real time.
5. Organizers can monitor attendance directly from the connected spreadsheet.
